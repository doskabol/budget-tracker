# 💰 Budget Tracker

A simple and intuitive web application to help you track your income, expenses, and manage your personal budget effectively. Built as a group project for web development course.

**Group Members:** Abdikarim Dosbol, Kystaubekov Mukhamedjan

---

🚀 Features

- ➕ Add income and expense transactions
- 📊 View real-time balance, total income, and total expenses
- 🗑️ Delete or edit transactions
- 📅 Filter transactions by date or category (optional if implemented)
- 📱 Fully responsive design (works on desktop, tablet, and mobile)
- 💾 Data persistence using `localStorage` (or backend if applicable)

---

🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend (optional):** Node.js / Express (if applicable)
- **Database (optional):** MongoDB / Firebase (if applicable)
- **Version Control:** Git & GitHub
